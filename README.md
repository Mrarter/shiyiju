# 拾艺局 Codex 开发启动包

## 1. 项目目标
“拾艺局”是一个面向微信生态的艺术品流通小程序，核心包含：
- 微信登录
- 艺术家主页
- 作品展示与动态定价
- 订单 / 支付 / 物流
- 收藏证书 / 托管
- 分销 / 团队 / 转售 / 拼团 / 盲盒 / 独家发行

## 3. 开发顺序
### Phase 1：主链路
1. 微信登录与用户体系
2. 艺术家主页
3. 作品列表/详情
4. 下单/订单
5. 发货/物流

### Phase 2：增长能力
1. 分销与团队
2. 收藏证书与托管
3. 转售市场

### Phase 3：高级玩法
1. 拼团收藏
2. 盲盒
3. 独家发行

## 4. 技术建议
- 后端：Spring Boot 3 + Java 17 + MyBatis + MySQL
- 小程序：微信原生小程序 / Taro（二选一）
- 文档规范：接口与字段以 docs/backend 为准
- 代码规范：统一响应体、统一异常处理、OpenAPI 注释必补

## 5. 给 Codex 的工作原则
- 一次只做一个模块
- 先读 docs，再改代码
- 每次提交输出：变更摘要 / 风险点 / 待联调项
- 不要擅自改接口风格与数据库命名
